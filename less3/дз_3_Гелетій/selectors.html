Какими бывают селекторы?

Номер по порядку Селектор Описание "Вес"
1 Теги html, body, div

Выбраны будет все элементы, которые находятся внутри данных елементов, за исключением тех, для которых заданы другие селекторы, например класс или id.

1
p, span, table, a
другие встроенные теги
любые придуманные теги
2 Классы ("class") Выбраны будут все элементы для которых задан соответствующий клас, за исключением тех, для которых заданы другие селекторы, например класс или id. 10
3 id Выбраны будут все элементы в заданном id. Данный селектор можно применить только один раз 100
4 style Как и id, в отличии от id - не задаётся через css-файл, инлайновый селектор 1000
5 Свойство "!important" Дополнительно свойство для повышения веса выбранного свойства элемента 10000
6 Комбинации селекторов Указывают особенность/последовательность применнения -
7 * Обращается ко всем элементам 0
